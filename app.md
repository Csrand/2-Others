# Desafio Dio - Gerenciador de Podcasts.

# Descrição

Um catologo de podcasts que permite exibir diferentes episodios, de categorias distintas a partir do gosto do usuário.

# Domínio

Podcasts feitos em vídeo.

# Features
- Listar os episódios em sessões de categorias.
- Filtrar episódios por nome de podcast.

# Como
vou retornar em uma api rest o nome do podcast, o titulo do vídeo junto com sua imagem e seu link de acesso.
```js
{
    podcastName: "falando nisso",
    episode: "Bipolaridade",
    videoId: "DUHjcS6wb0g",
    cover: "https://i.ytimg.com/vi/DUHjcS6wb0g/maxresdefault.jpg",
    link: "https://www.youtube.com/watch?v=DUHjcS6wb0g",
    categories: ["psicanalise", "perguntas e respostas"]
}
```
