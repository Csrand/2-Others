# 2-Others
Criei este repositório para os "Outros". Nele vou focar em demonstrar domínio em demandas reais do mercado, que observo com frequência em descrições das vagas de trabalho.

# Organização
Vou separar o repositório em 3 áreas: front, back e deploy. Em cada uma delas vou explorar os tópicos mais recorrentes em vagas de emprego para desenvolvedores juniores no mercado brasileiro.
